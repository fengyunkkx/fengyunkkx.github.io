Windows 方案

操作步骤：

1.下载带配置文件的 Windows 版 V2RayW

2. 解压后打开 V2RayW.exe。

3. 右下角会多出一个图标「右键图标 - 加载 V2Ray」，启动服务。

完成，打开 https://twitter.com/ 测试。

补充：

默认使用自动模式（PAC），会自动识别是否需要代理，节省流量。如果需要让所有流量都走代理，则选择全局模式。

macOS 方案

操作步骤：

1. 下载 macOS 版 V2RayX

2. 解压后，将 V2RayX.app 移动到应用程序中安装。

3. 启动 V2RayX，点击右上角图标 - Configure，添加服务器配置。

4. 在设置界面点击 ＋ 按钮，在右侧添加服务器配置。点击 OK 完成添加。

5. 重新点击右上角图标 - Load core，启用服务。

完成，打开 https://twitter.com/ 测试。

补充：

默认使用 V2Ray Rules，自动识别是否需要代理，节省流量。如果需要让所有流量都走代理，选择 Global Mode。
